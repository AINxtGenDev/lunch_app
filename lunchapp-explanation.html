🍜 Lunch Menu App

Automated restaurant menu scraping and display system

Version 2.3 - Production Ready

🎯 Project Overview

The Lunch Menu App is a sophisticated web application that automatically scrapes and displays daily menus from multiple local restaurants. Built with modern web technologies and deployed on Raspberry Pi 4 for reliable 24/7 operation.

🔧 Raspberry Pi Setup

Environment Setup

# Create conda environment conda env create -f environment.yaml conda activate lunch-menu-app # Initialize database python init_db.py

Production Configuration

# Update .env for production FLASK_ENV=production DATABASE_URL=sqlite:////home/stecher/lunch_app/instance/app.db # Configure Gunicorn workers = 1 worker_class = "eventlet" bind = "0.0.0.0:7000"

Systemd Service Setup

# Copy service file sudo cp lunch-app.service /etc/systemd/system/ # Enable auto-start sudo systemctl enable lunch-app sudo systemctl start lunch-app

Reverse Proxy with Caddy

# Configure Caddy for HTTPS mealplanchecker.duckdns.org:10557 { tls otto.kuegerl@gmail.com reverse_proxy localhost:7000 } # Reload configuration sudo systemctl reload caddy
